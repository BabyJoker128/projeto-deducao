# 🔍 Projeto Dedução

Uma aplicação web voltada para experiências de jogos investigativos, permitindo a gestão de pistas e o progresso de deduções em tempo real.

## 🛠️ Desafios Técnicos Superados
* **Configuração de CORS:** Implementação de lógica robusta para permitir a comunicação segura entre o frontend e o backend hospedados em plataformas diferentes (Vercel e Render).
* **Gestão de Assets:** Sincronização e filtragem dinâmica de arquivos e layouts de investigação.

## 🚀 Tecnologias e Ferramentas
* **Frontend:** Vite + React
* **Backend:** Lógica de servidor para protocolos de investigação.
* **Deploy:** Vercel (Frontend) e Render (Backend).

## 🔗 Link da Aplicação
Teste o projeto aqui: [https://projeto-deducao.vercel.app/](https://projeto-deducao.vercel.app/)

## ⚙️ Como executar o projeto localmente
1. Clone o repositório.
2. Instale as dependências com `npm install`.
3. Execute o comando `npm run dev`.
